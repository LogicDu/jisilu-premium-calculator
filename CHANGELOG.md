# 更新日志

所有重要的项目变更都会记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，

## [1.3.0] - 2026-03-11

### 新增
- ✨ 支持QDII基金页面 (`/data/qdii/*`)
- ✨ 动态识别当前激活的表格（根据URL hash）
- ✨ 支持多表格页面切换时自动重新初始化

### 改进
- 🔧 重构代码架构，支持多页面配置
- 📝 更新项目名称为 `jisilu-premium-calculator`
- 📝 更新GitHub仓库地址

## [1.2.0] - 2026-03-11

### 新增
- ✨ 添加点击表头排序功能（降序→升序→取消）
- ✨ 排序状态指示器（↑/↓箭头）

### 修复
- 🐛 修复表头悬浮时溢价率数据显示在表头之上的问题
- 🐛 添加 `sticky` 类和正确的 `z-index` 样式

### 改进
- 🎨 添加表头悬停效果
- 🎨 优化排序逻辑

## [1.1.0] - 2026-03-11

### 修复
- 🐛 **关键修复**：修复脚本查找错误表格ID导致初始化超时的问题
  - 原因：脚本只查找 `#flex_index`，但页面根据hash显示不同表格
  - 解决：根据URL hash动态选择正确的表格ID

### 新增
- ✨ 支持LOF页面的三个表格：`#flex_index`、`#flex_stock`、`#flex_arb`
- ✨ 监听hash变化，切换表格时自动重新初始化

### 改进
- 🔧 检查表格可见性，只处理当前显示的表格
- 📝 更新版本号和日志

## [1.0.2] - 2026-03-11

### 修复
- 🐛 修复表格ID错误：从 `index_lof_table` 改为 `flex_index`

- 🐛 修复单元格定位：使用列索引而非data-field属性
- 🐛 修复脚本重复代码问题

### 改进
- ✨ 添加更详细的调试日志
- ✨ 改进表格检测逻辑

### 测试
