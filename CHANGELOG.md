## [1.4.0] - 2026-03-12

### 新增
- ✨ **实时估值功能**：新增"实时估值"列，显示基金的盘中实时估值
  - 数据来源：天天基金网公开API
  - 估值时间：交易日9:30-15:00实时更新
  - 缓存机制：60秒缓存避免重复请求

### 改进
- 🔧 **溢价率计算优化**：现在使用实时估值计算溢价率，更加准确
  - 之前：溢价率 = (场内价 - T-1日净值) / T-1日净值
  - 现在：溢价率 = (场内价 - 实时估值) / 实时估值
- 🔧 添加数据获取失败处理，显示 "-" 标记
- 🔧 异步数据加载，不阻塞页面渲染

### 技术细节
- 📦 新增 `fetchRealTimeEstimate()` 异步函数获取实时估值
- 📦 新增 `estimateCache` 缓存机制
- 📦 修改 `addPremiumDataToRow()` 为异步函数
- 📦 修改 `processAllRows()` 为异步函数
- 📦 调整溢价率列索引（navIndex + 2）

### 背景
- 集思录已于2026-03-08停止提供基金估值数据
- 本次更新通过第三方API恢复实时估值功能

### 修复

- 🐛 **关键修复**：修复 QDII 页面"商品"表格不显示溢价率列的问题
  - 原因：QDII 页面的 `#qdiie` hash 下有两个表格（欧美指数和商品），但脚本只处理了第一个表格
  - 解决：修改配置结构支持一个 hash 对应多个表格，并遍历初始化所有表格
  - 相关 Issue: [#1](https://github.com/LogicDu/jisilu-premium-calculator/issues/1)
