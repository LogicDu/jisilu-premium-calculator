# 常见问题解答（FAQ）

## 安装相关

### Q1: 我应该使用哪个浏览器？

**推荐使用 Chrome 或 Edge 浏览器**，这两个浏览器对Tampermonkey的支持最好，性能也最优。

支持的浏览器：
- ✅ Chrome 90+ （推荐）
- ✅ Edge 90+ （推荐）
- ✅ Firefox 88+
- ✅ Safari 14+

### Q2: 安装后没有看到溢价率列怎么办？

请按以下步骤排查：

1. **检查脚本是否启用**
   - 点击Tampermonkey图标
   - 确认"集思录LOF溢价率计算"前面是绿色圆点
   - 如果是灰色，点击切换为启用

2. **检查页面URL**
   - 确认访问的是：`https://www.jisilu.cn/data/lof/`
   - 脚本只在LOF页面生效

3. **刷新页面**
   - 按 F5 或点击刷新按钮
   - 等待页面完全加载

4. **查看控制台**
   - 按 F12 打开开发者工具
   - 切换到 Console 标签
   - 查看是否有错误信息

## 使用相关

### Q4: 为什么有些基金显示"--"？

显示"--"表示无法计算溢价率，可能的原因：

1. **净值数据缺失**
   - 基金净值字段为空
   - 等待数据更新

2. **新上市基金**
   - 刚上市的基金可能还没有净值数据
   - 通常T+1会有数据

3. **数据格式异常**
   - 价格或净值不是有效数字
   - 这种情况较少见

### Q5: 溢价率的正负号是什么意思？

- **正溢价（+）**：场内价格 > 净值
  - 例如：`+1.50%` 表示场内价格比净值高1.5%
  - 可能存在套利空间（卖出场内，申购场外）

- **负溢价（-）**：场内价格 < 净值
  - 例如：`-0.30%` 表示场内价格比净值低0.3%
  - 可能是买入机会（买入场内，赎回场外）

### Q6: 溢价率不更新怎么办？

1. **手动刷新**
   - 点击页面上的"手动刷新"按钮
   - 或按 F5 刷新整个页面

2. **检查自动刷新**
   - 如果开启了"30秒自动刷新"
   - 溢价率会随表格数据自动更新

3. **重新加载脚本**
   - 禁用脚本后重新启用
   - 或刷新页面

### Q7: 溢价率计算准确吗？

脚本使用的计算公式为：

```
溢价率 = (场内实时价 - 基金净值) / 基金净值 × 100%
```

这是标准的溢价率计算方法，计算结果准确可靠。

**注意事项：**
- 场内价格是实时的
- 基金净值通常是T-1日的数据
- 存在时间差，仅供参考

## 自定义相关

### Q11: 如何修改溢价率的颜色？

1. 点击Tampermonkey图标 → 管理面板
2. 找到"集思录LOF溢价率计算"，点击"编辑"
3. 找到 `CONFIG` 对象：

```javascript
const CONFIG = {
    COLUMN_NAME: '溢价率',
    COLUMN_WIDTH: '80px',
    POSITIVE_COLOR: '#ff4444',  // 正溢价颜色
    NEGATIVE_COLOR: '#00aa00',  // 负溢价颜色
    DECIMAL_PLACES: 2,
};
```

4. 修改颜色值（使用十六进制颜色代码）
5. 保存（Ctrl+S 或 Cmd+S）
6. 刷新页面

**常用颜色：**
- 红色：`#ff0000`, `#ff4444`, `#e74c3c`
- 绿色：`#00ff00`, `#00aa00`, `#27ae60`
- 蓝色：`#0000ff`, `#3498db`, `#2980b9`

### Q12: 如何修改小数位数？

修改 `CONFIG` 中的 `DECIMAL_PLACES`：

```javascript
DECIMAL_PLACES: 3,  // 显示3位小数
```

保存后刷新页面即可。

### Q13: 可以修改列名吗？

可以，修改 `CONFIG` 中的 `COLUMN_NAME`：

```javascript
COLUMN_NAME: '溢价',  // 或其他你喜欢的名称
```
